    QQ Music

    • QQ 音乐移动端单页应用页面，包括推荐页、排行榜页以及搜索页的实现，在搜索音乐后点击单曲进行播放。

  • 003

    豆瓣租房爬虫

    • 这是一个针对豆瓣杭州租房小组的爬虫，爬取豆瓣杭州租房小组帖子信息，输入需要查询的地段，查询到信息后展示到前端页面。
    • nodejs 部分使用 axios 发送请求，使用 cheerio 对请求结果进行处理，使用 express 搭建服务器。
    • 由于豆瓣对外链的限制以及反爬虫策略，导致无法展示图片，当请求过于频繁时会被封 IP 。

    部署： Express + Nodejs

    技术栈： React + webpack + jQuery + Cheerio + Nodejs + Express

  • 004

    React 照片墙

    • 这是一个 React 照片墙的随机排布展示。
    • 点击中心以外的图片该图片会置于中心，点击中心图片可以进行翻转
    • 点击底部导航按钮会展示对应的图片，并将其置于中心，再次点击导航按钮将图片进行翻转
    • 图片动画效果采用CSS3 动画属性transform、transition等
    • 使用less-loader、style-loader、url-loader等工具自动编译 less文件、图片以及 iconfont 文件

    技术栈： React + webpack + ES6 + CSS3 + less
